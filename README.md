## Supported Features 

► Swipe Videos                                                            
► Dynamic Video Data Source                                                                  
► User Data (Name, Image)                                         
► Login with Google                            
► See videos                                              
► Like                                                            
► comment                                
► messaging                                   
► Follow                                                             
► unfollow                                                            
► Record new video                                                                                                 
► Upload clips                               
► TikTok UI
